# Поиск самой длинной подстроки без повторений

## Описание проекта
Этот проект содержит реализацию алгоритма для поиска самой длинной подстроки без повторяющихся символов в строке. Проект состоит из трех основных функций, которые работают вместе для обработки списка произвольных элементов.

## Функциональность
Программа выполняет следующие задачи:
1. Анализирует входной список, содержащий элементы любых типов
2. Находит самую длинную строку в списке
3. В найденной строке находит самую длинную подстроку без повторяющихся символов
4. Возвращает индекс найденной строки и найденную подстроку

## Структура кода

### Функции

#### 1. `check_input(element)`
- **Назначение**: Проверяет, является ли переданный элемент строкой
- **Вход**: Любой элемент
- **Выход**: `True` если элемент строка, `False` в противном случае
- **Использует**: `isinstance(element, str)`

#### 2. `get_substring(s)`
- **Назначение**: Находит самую длинную подстроку без повторяющихся символов
- **Вход**: Строка для анализа
- **Выход**: Самая длинная подстрока без повторений
- **Алгоритм**: Использует метод "скользящего окна" с временной сложностью O(n)

#### 3. `main(lst)`
- **Назначение**: Основная функция, координирующая работу программы
- **Вход**: Список произвольных элементов
- **Выход**: Кортеж (индекс_строки, найденная_подстрока)
- **Особенности**: Возвращает `(-1, '')` если в списке нет строк
